# Module 1 Task Tracker API

A minimal Python FastAPI REST API for a learning-project Task Tracker application.

This project uses a simple layered architecture:

- API routes
- Service layer
- JSON storage layer

For now, the skeleton only includes a health check endpoint. CRUD endpoints will be added later.

## Requirements

- Python 3.10 or higher
- pip
